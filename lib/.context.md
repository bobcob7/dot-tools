# lib

## Purpose

Shared shell library functions used by setup scripts and modules.

## Functionality

- Colored logging functions (info, success, warn, error)
- OS detection (macOS, Ubuntu, Linux)
- Package installation abstraction (Homebrew/apt)
- File operations (backup, symlink with safety)
- Temporary workspace creation (playground)

## Files

| File | Description |
|------|-------------|
| `helpers.sh` | Core utility functions for logging, OS detection, package management, and file operations |
| `playground.sh` | Functions to create and manage temporary workspaces in /tmp/playground |
