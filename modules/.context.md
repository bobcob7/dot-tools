# modules

## Purpose

Individual setup modules for different tools. Each module handles installation and configuration of a specific tool or category of tools.

## Functionality

- Standalone setup scripts that can be run individually or via setup.sh
- Each module provides a `setup_<name>` function as entry point
- Modules are idempotent (safe to run multiple times)

## Files

| File | Description |
|------|-------------|
| `zsh.sh` | Installs zsh, Oh-My-Zsh, plugins (autosuggestions, syntax highlighting), and Dracula theme |
| `tmux.sh` | Installs tmux, Oh My Tmux, and configures Dracula-themed status bar |
| `vim.sh` | Installs Vim, vim-plug, plugins (LSP, fzf, NERDTree), and Dracula colorscheme |
| `utils.sh` | Installs utilities (Python, pip, gh CLI) and copies scripts to ~/.local/bin |
| `claude.sh` | Installs Claude Code CLI, syncs user configuration, and installs plugins |
