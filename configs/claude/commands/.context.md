# commands

## Purpose

Claude Code slash commands (`/command-name`) that get synced to `~/.claude/commands/` via `sync.sh`. These are markdown prompt files that Claude executes when invoked.

## Functionality

- Provides reusable slash commands for Claude Code
- Synced as `dir-md` type (flat markdown files)

## Files

| File | Description |
|------|-------------|
| `update-context.md` | Scan, validate, and update directory context via sudo-context MCP tools using parallel subagents grouped by depth |
