# PlantUML MCP Server

## Purpose

MCP server that encodes PlantUML diagrams and generates URLs for the public PlantUML server.

## Functionality

Provides a single tool `plantuml_generate_urls` that returns URLs for:
- PNG image
- SVG image
- TXT (ASCII art)
- Online editor with diagram preloaded

## Files

| File | Description |
|------|-------------|
| `main.py` | MCP server implementation (pure Python 3, stdlib only) |
