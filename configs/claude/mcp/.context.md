# MCP Servers

## Purpose

Contains Model Context Protocol (MCP) servers that extend Claude's capabilities with custom tools. Each subdirectory is a self-contained MCP server.

## Functionality

- PlantUML diagram URL generation via the `plantuml_generate_urls` tool

## Files

| Directory | Description |
|-----------|-------------|
| `plantuml/` | MCP server for generating PlantUML image and editor URLs |
