# skills/plan-status

## Purpose

Skill for checking implementation plan progress. Reads `plans/.context.md` to report which plans are done, which are ready, and what to implement next.

## Functionality

- Reports overall plan completion progress
- Identifies plans ready to implement (all dependencies satisfied)
- Identifies blocked plans and their unmet dependencies
- Recommends the next plan to work on based on priority and dependency order
