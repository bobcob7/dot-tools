# skills/updated-context

## Purpose

Skill for validating, creating, and incrementally updating `.context.md` files across a codebase using git diff to detect staleness.

## Functionality

- Scans directories for missing or stale context files
- Detects staleness by comparing stored git ref against current HEAD
- Uses `git diff` for targeted incremental updates of changed sections
- Generates full context files for directories missing them
- Validates existing context files have required sections
