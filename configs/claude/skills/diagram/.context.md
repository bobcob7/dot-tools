# diagram

## Purpose

Claude Code skill for creating PlantUML diagrams and embedding them in documentation. Uses the PlantUML MCP server to generate image URLs.

## Functionality

- Create sequence, component, class, activity, and state diagrams from natural language descriptions
- Generate PlantUML PNG/SVG/editor URLs via the plantuml MCP plugin
- Embed diagrams as clickable linked images in markdown files
