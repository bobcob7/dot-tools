# update-context

## Purpose

Claude Code skill for scanning, validating, and updating directory context using the sudo-context MCP server. Replaces the previous `.context.md`-based update-context skill with structured JSON storage.

## Functionality

- Scan all project directories for missing, stale, or orphaned context
- Create context for new directories with all 8 required sections
- Update stale context by diffing changes since last recorded git ref
- Report findings to the user with actionable summaries
