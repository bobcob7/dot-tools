# skills/plan-create

## Purpose

Skill for creating agent-executable implementation plans. Generates a `plans/` directory with `.context.md` tracking and individual plan files following a consistent format.

## Functionality

- Decomposes features into single-agent-session plans
- Creates `plans/.context.md` with dependency graph, files table, and TODO tracking
- Creates individual plan files with context, scope, code snippets, verification, and acceptance criteria
- Creates meta-plans for complex features with sub-plan breakdowns
- Plans are compatible with `/plan-status` and `/plan-loop` skills
