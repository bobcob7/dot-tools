# configs/claude

## Purpose

User-level Claude Code configuration. Synced to ~/.claude/ during setup.

## Functionality

- Global user preferences and coding style guidelines
- Permission allowlists and denylists for Claude Code tools
- Reusable skills, agents, and rules for consistent behavior
- Custom MCP servers for extending Claude's capabilities

## Files

| File/Directory | Description |
|----------------|-------------|
| `CLAUDE.md` | User preferences for code style, git workflow, and language-specific conventions |
| `settings.json` | Permission configuration allowing/denying specific tool patterns |
| `mcp/` | MCP server implementations (registered via `claude mcp add`) |
| `skills/` | Reusable skills (workflows) for Claude |
