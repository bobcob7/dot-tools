# configs/claude/agents

## Purpose

Custom agent definitions for Claude Code's Task tool. Agents are specialized subprocesses with focused expertise.

## Functionality

- Language-specific expert agents (Go, Rust, TypeScript, JavaScript, HTML/CSS)
- Task-specific agents (code review, test running, documentation)
- Each agent has defined tools and model preferences

## Files

| File | Description |
|------|-------------|
| `code-reviewer.md` | Agent for thorough code review focused on correctness, security, and maintainability |
| `test-runner.md` | Agent for running tests and analyzing failures |
| `go-expert.md` | Go language specialist for idiomatic Go patterns |
| `rust-expert.md` | Rust specialist for safe, performant code |
| `typescript-expert.md` | TypeScript specialist for type-safe development |
| `javascript-expert.md` | JavaScript specialist for modern ES6+ development |
| `html-css-expert.md` | HTML/CSS specialist for semantic markup and styling |
| `markdown-expert.md` | Markdown and MermaidJS specialist for documentation |
