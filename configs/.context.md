# configs

## Purpose

Configuration files to be installed or symlinked to user home directories.

## Functionality

- Shell configuration (zsh customizations)
- Editor configuration (Vim)
- AI assistant configuration (Claude Code)

## Files

| File | Description |
|------|-------------|
| `zshrc.local` | Custom zsh configuration sourced by .zshrc (aliases, history settings, tmux auto-start) |
| `vimrc` | Complete Vim configuration with plugins, LSP, and Dracula theme |
